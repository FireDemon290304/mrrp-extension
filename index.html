Privacy Policy for :3 Sound Player Extension

Information Collection

  • No personal data is collected
  • No browsing history is stored
  • No cookies are used
  • Audio files are cached locally in browser memory

Data Usage

  • Text content is temporarily analyzed for ":3" emoticons
  • DOM modifications are made only for smiley detection
  • Changes are reversed when sound is disabled
  • No data transmission occurs

Security Measures

  • HTTPS connections only
  • Local audio caching
  • Memory cleanup on sound disable
  • No external data storage
